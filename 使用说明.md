# 监装报销统计 · 使用说明

## 这是什么

一个记录船舶监装报销与费用的电子台账。它是一个网页文件,双击就能用,不用安装任何软件、不用联网。你在原来 Excel 里做的事,现在都能在这里做,而且人员汇总、船东汇总、利润统计会自动算好。

## 怎么打开

双击 `监装报销统计_20260715_v1.7.1_gpt.html`(或你收到的当前版本文件名),会用浏览器(Edge 或 Chrome)打开。

## ⚠️ 最重要的一件事:数据存在哪

**你的数据保存在这台电脑的浏览器里,不会存进这个网页文件本身。**

这意味着:

- 在**同一台电脑**上,不管把这个文件放哪个文件夹、改什么名字,打开看到的都是同一份数据,数据不会丢。
- 但如果你把这个文件**发给别人、或换一台电脑打开**,数据不会跟着文件走——对方看到的是最初的历史数据,不是你后来记的账。
- **换电脑 / 备份 / 传给别人时**:在「设置与数据」里点「导出 JSON 备份」存成一个文件,到新环境里点「导入」,数据就过去了。

**建议每隔一段时间导出一次备份**(JSON 或 Excel 都行),存到网盘或 U 盘,万一电脑出问题不至于丢数据。软件超过 7 天没导出会主动提醒你。

## 五个页签怎么用

1. **台账流水**:日常记账的主页。点「+ 记一笔」填一条报销;可以按船次、人员、类型筛选查看。表格很宽需要左右滚动时,**「日期」和「船名」两列会固定在左边不动**,滚到右边看利润、操作时也不会认错是哪条船;每条新船次的第一行左边有一道竖色条,方便区分不同船次。
2. **船次管理**:管理每一个船次(船名、船号、船东、监装天数、汇率、监装费)。点「+ 新建船次」。
3. **按人员**:每个人的报销、补贴、天数自动汇总;可以点开看明细。
4. **按船东**:每个船东的监装费、利润自动汇总。
5. **设置与数据**:管理人员名录、船东名录、报销科目、汇率清单;人员序号使用 `001` 起的三位数字且不可重复;导出、导入、备份都在这里。

## 记账时的几个说明

- **报销科目**:常规 / 住宿费 / 机票 三类,不够用可以在设置里自己加。
- **美元支付**:在「记一笔」里选择科目「美元支付」,填写美元原值和汇率。系统折算为人民币保存;这类钱不算进"人民币费用",但算进船次总支出。
- **监装费**:填了日单价和天数会自动算,也可以手动改成任意金额(改过会显示"手工"标记,点一下能恢复自动)。
- **汇率**:在设置里维护常用汇率,记账时下拉选;纯人民币的船次可以留空。
- **填表日期**:记一笔时默认是今天,可以改,也可以留空;补录旧账时记得改成实际日期。

## 导出给别人 / 上报

点「导出 Excel」,生成一个 Excel 文件(报销统计表 + 按人员 + 按船东三张表),可以直接发给别人或打印上报。「报销统计表」会从船号中提取年份、月份和次数自然升序排列,示例船号按数字核心排序;同一船次的流水保持连续。主表的「监装人员」和「科目」是独立两列,方便筛选和透视。「按人员汇总」会按人员序号升序排列,未编号人员排在最后。表格末尾的「利润总计」行左边已经合并成一格;主表里两个"天数"列分别叫「流水天数」和「船次监装天数」,不会混。

导出表格式(v1.7.1):全文 **Times New Roman 9 号**;表头/正文/合计行高分别为 20/16/18,列宽按 9 号字体和字段类型自动收紧;人民币金额显示 `¥`,「监装费USD」显示 `$`,金额仍是可求和的数值;「按船东汇总」末行会对船次数、天数和各金额列逐列合计;表头加粗、浅蓝底并居中,合计行浅琥珀底;不同船次使用极浅底色和分组线区分,人名与科目只在各自单元格内使用淡色标识,负利润使用浅红预警;表格使用细边框、隐藏网格线;金额/天数/汇率右对齐便于对数,其余文本居中;单元格不自动换行;三张表开启筛选;「报销统计表」冻结首行和前两列(船名、船号),另外两张表冻结首行。

导出成功后,右上角会短暂显示「✓ 刚刚已导出」;如果长时间没导出,顶部会有黄色提醒条,条上直接有「立即导出备份」按钮可以点。

## 遇到问题

- **打开后是空白或报错**:换用 Edge 或 Chrome 打开(不要用很老的 IE)。
- **提示"保存失败/数据仅在内存"**:说明这个浏览器环境不让存数据,请立刻导出备份,并换一个环境打开。
- **数据好像不见了**:先别慌,大概率是换了电脑或浏览器(见上面"数据存在哪")。用之前导出的备份「导入」即可找回。

## v1.7.2 新增(2026-07-15)

- **多窗口打开警告**:如果你不小心把台账在同一台电脑上打开了两个(或更多)窗口/标签页,每个窗口顶部都会出现一条醒目的红色警告:「检测到台账在多个窗口/标签页同时打开」。**两个窗口同时录入会互相覆盖数据**,看到这条警告请立即关闭多余的窗口,只保留一个使用。警告不用手动关——多余的窗口关掉后,过十几秒它会自己消失。这只是提醒,不会锁住或限制任何功能,请自觉遵守"一次只开一个窗口"。
- **从旧版本升级后请注意**:升级到 v1.7.2 后,请**关闭旧版本的页面、不要再打开旧版文件使用**——旧版本(v1.7.1 及更早)没有这个多窗口检测,新旧版本同时开着时旧版窗口不会弹任何警告,互相覆盖数据的风险更大。日常只用最新版这一个文件即可。
- **窄屏幕提示**:这个工具是按电脑宽屏设计的。如果把窗口拖得很窄(小于 1100 像素宽,比如缩成手机那么窄),顶部会出现一条黄色提示,建议把窗口拉宽一点以免表格显示不全;这只是提醒,不影响任何功能,点右边的「×」可以关掉,拉宽窗口后会自动消失。
- **键盘操作更方便**:弹窗(如「记一笔」「新建船次」「删除确认」等)现在可以按 `Esc` 键直接关闭,不用每次都点鼠标「取消」;顶部的「台账流水/船次管理/按人员/按船东/设置与数据」五个页签,用键盘左右方向键就能切换。
- **界面响应更快**:船次多、流水多以后,「船次管理」「按船东汇总」等页面的加载和刷新做了内部优化,数据不变、看起来会更跟手一些,不需要你做任何操作。
